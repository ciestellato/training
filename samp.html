VSCodeの使い方

HTMLファイルの作成

  1. htmlファイルを作成し、保存する
  2. 何も記述していないプレーンな状態で「!」（半角エクスクラメーションマーク） を入力する
  3. サジェストが表示されるので、Tabキーで選択する
  4. HTMLページに必要なコードが記述される

便利な使い方

<>の省略 「td」等各種タグをカッコなしで直接打ち込み、Enterを押すと タグとして入力が補完され、カーソルは間の部分に移動する
Ctrl + Enter 行の途中でCtrl+Enterを押すと、その行の後の入力はそのままで カーソルが次の行へ移動する
Ctrl + 左右 行頭や行末にカーソルを移動する
Ctrl + 上下 カーソルはそのままで画面を1行ずつスクロールする
