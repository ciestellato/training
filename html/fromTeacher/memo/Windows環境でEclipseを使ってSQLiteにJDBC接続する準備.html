Windows環境でEclipseを使ってSQLiteにJDBC接続するには、以下の手順でセットアップできます。 とても軽量で、ローカル開発にぴったりな構成です。

🧰 ステップ1：SQLiteのインストール（またはDBファイル作成）

SQLiteはインストール不要ですが、公式ツールを使うと便利です。

  1. SQLite公式ページから「sqlite-tools-win32-x86」をダウンロード
  2. ZIPを解凍し、sqlite3.exe を任意のフォルダに配置
  3. コマンドプロンプトで以下を実行してDBファイルを作成：
sqlite3 test.db

これで test.db というSQLiteファイルが作成されます。

SQLite3 よく使うコマンド集

📦 ステップ2：JDBCドライバーのダウンロード

  1. Xerial SQLite JDBC Driver にアクセス
  2. 最新の .jar ファイル（例：sqlite-jdbc-3.43.2.0.jar）をダウンロード

🖥️ ステップ3：EclipseプロジェクトにJDBCドライバーを追加

  1. EclipseでJavaプロジェクトを作成
  2. プロジェクトを右クリック → Build Path → Configure Build Path
  3. Libraries タブ → Add External JARs → ダウンロードした .jar を選択
  4. Apply and Close を押して完了

🔌 ステップ4：SQLiteに接続するJavaコード例

import java.sql.*;

public class SQLiteExample {
    public static void main(String[] args) {
        String url = "jdbc:sqlite:C:/sqlite/test.db";  // DBファイルの絶対パス

        try (Connection conn = DriverManager.getConnection(url)) {
            System.out.println("接続成功！");

            String sql = "CREATE TABLE IF NOT EXISTS expenses (id INTEGER PRIMARY KEY, category TEXT, amount INTEGER, spent_on TEXT, memo TEXT)";
            try (Statement stmt = conn.createStatement()) {
                stmt.execute(sql);
                System.out.println("テーブル作成完了");
            }
        } catch (SQLException e) {
            System.out.println("接続エラー: " + e.getMessage());
        }
    }
}

✅ 補足ポイント

  • SQLiteのJDBC URLは jdbc:sqlite:ファイルパス
  • ファイルパスは 絶対パスが確実（Windowsなら C:/...）
  • Eclipseで実行する際は、JDKのバージョンとJARの互換性に注意

これで、EclipseからSQLiteに接続してJDBC経由で操作する準備は完了です！次はデータのINSERTやSELECTなど、家計簿アプリのロジックを組み立てていくフェーズです。
