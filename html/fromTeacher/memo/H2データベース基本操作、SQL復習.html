H2データベースを使って、employee（従業員）とdept（部署）テーブルを操作する基本的なSQL文を、初学者向けに分かりやすくまとめました。各ステートメントには簡単な解説も付けています。

🔍 参考サイト:

  • H2 Database インストールから基本操作まで https://qiita.com/YJ2222/items/ca239cb20e0e88463571
  • H2 Database（H2DB）におけるテーブル作成方法 詳細チュートリアル https://qiita.com/jtths474/items/7ec969bb77bfc871b0ab

🏗️ 1. テーブル作成（CREATE TABLE）

CREATE TABLE dept (
  dept_id INT PRIMARY KEY,
  dept_name VARCHAR(50)
);

CREATE TABLE employee (
  emp_id INT PRIMARY KEY,
  emp_name VARCHAR(50),
  salary INT,
  dept_id INT,
  FOREIGN KEY (dept_id) REFERENCES dept(dept_id)
);

🔍 解説:

  • dept は部署テーブル。dept_id が主キー。
  • employee は従業員テーブル。dept_id は部署との関連を示す外部キー。

📝 2. データ挿入（INSERT）

INSERT INTO dept VALUES (1, '営業部');
INSERT INTO dept VALUES (2, '開発部');

INSERT INTO employee VALUES (101, '田中太郎', 500000, 1);
INSERT INTO employee VALUES (102, '佐藤花子', 600000, 2);

🔍 解説:

  • dept に2つの部署を追加。
  • employee に2人の従業員を追加。部署IDで紐づけ。

🔍 3. データ取得（SELECT）

SELECT * FROM employee;

🔍 解説:

  • employee テーブルの全データを表示。

✏️ 4. データ更新（UPDATE）

UPDATE employee
SET salary = 650000
WHERE emp_id = 102;

🔍 解説:

  • emp_id が102の佐藤花子さんの給与を更新。

🗑️ 5. データ削除（DELETE）

DELETE FROM employee
WHERE emp_id = 101;

🔍 解説:

  • emp_id が101の田中太郎さんのデータを削除。

🔗 6. テーブル結合（SELECT INNER JOIN）

SELECT e.emp_name, e.salary, d.dept_name
FROM employee e
INNER JOIN dept d ON e.dept_id = d.dept_id;

🔍 解説:

  • employee と dept を部署IDで結合。
  • 従業員名・給与・部署名を一覧表示。

💡補足：H2データベースでの実行方法

H2コンソールやJavaアプリから以下のように接続できます：

Connection conn = DriverManager.getConnection("jdbc:h2:~/test", "sa", "");

その後、上記SQLを Statement や PreparedStatement で実行できます。
